# Weft

Weft is how you turn a rough idea into a product you own. A team of specialists walks every step with you, from the first question to a real product.

Weft guides you through every stage of building software — strategy,
design, engineering, QA, security, deployment — so you ship something
real instead of a pile of unusable AI-generated code.

It is a **Claude Code plugin**. One command: `/weft`. It reads where
you are and does the right next thing.

---

## How it works

`/weft` is a state-aware router. It runs the Orchestrator, which
drives a structured pipeline of specialized stages with a human
sign-off at every real decision point:

- **New feature** — Strategy → PRD → Schema → Design → Build → QA →
  Security → Deploy → Record
- **Improve** / **Fix** — focused lanes for changing existing code

Infrastructure is provisioned just-in-time, at the stage that needs
it — never an upfront setup wall. The design system is established
the first time you see your product and reused by every feature
after, so it stays coherent.

## Install

Weft runs inside Claude Code (CLI, desktop, or an IDE) — not the
Claude chat app or claude.ai/code. You need a project folder open.

Hosted distribution is coming soon via the Anthropic plugin
marketplace and at [weft.build](https://weft.build).

## One-time, no subscription

Weft is a one-time download. No tiers, no recurring cost — it runs
entirely on your own Claude subscription, so it costs nothing extra
per use.
